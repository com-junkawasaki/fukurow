🔧 技術仕様

  • 言語: Rust → WebAssembly (wasm-bindgen)
  • 推論: OWL Lite/DL (Tableau Algorithm)
  • ストレージ: インメモリRDFトリプルストア
  • 可視化: HTML5 Canvas API
  • パース: Turtle/JSON-LD対応
  • 性能: ブラウザネイティブ速度

📚 使用例

  1. RDFデータを入力欄に貼り付ける
  2. 「RDFデータを読み込む」ボタンをクリック
  3. 「整合性チェック」ボタンでオントロジーの整合性を検証
  4. Canvas上で知識グラフの構造を確認
